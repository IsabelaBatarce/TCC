O que é o Moodle?

A sigla Moodle, é originalmente um acrônimo para Modular Object-Oriented Dynamic Learning Environment (Ambiente de Aprendizagem Dinâmico Modular Orientado a Objeto) e a plataforma se resume em um sistema de gerenciamento de aprendizagem para criação de cursos online e salas de aula virtuais.

Tema Boost

O Bitnami automaticamente instala o moodle com o tema boost, e este guia serve para alterar apenas este tema. Os arquivos acimas estão localizados em : C:\Bitnami\moodle-3.5.2-1\apps\moodle\htdocs\theme\boost\templates

Arquivo Preset

O arquivo default e/ou plain possuem todas as variáveis scss da plataforma, caso queira mudar o estilo, este são os arquivos principais, localizado em: C:\Bitnami2\moodle-3.5.2-1\apps\moodle\htdocs\theme\boost\scss\preset

Vídeo explicativo

Para que a modificação do css feita no código-fonte seja aplicada na página, são necessários os passos descritos no vídeo.

Como alterar o CSS da página?

As principais pastas para alteração do estilo estão localizadas em : C:\Bitnami2\moodle-3.5.2-1\apps\moodle\htdocs\theme\boost\scss.

O que é o Bitnami?

Uma aplicação AMP que transforma o computador em um servidor web. Outrossim, existe um pacote específico, o Bitnami Moodle Stack, que após ser instalado, realiza a hospedagem da plataforma Moodle no computador.
